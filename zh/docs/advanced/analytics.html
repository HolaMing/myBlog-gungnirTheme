站点统计

Richer
2021-03-26
1 min

信息

默认不启用这些功能。

提示

对于没有启用的功能，与它们相关的代码并不会被打包。

目前支持谷歌统计（基于插件 plugin-google-analytics）和百度统计（基于插件 plugin-baidu-tongji）。

# 谷歌统计

需要去谷歌统计官网上申请一个跟踪代码，然后填入配置：

theme: gungnirTheme({
  themePlugins: {
    ga: "your-tracking-code"
  }
})

# 百度统计

需要去百度统计官网上申请一个跟踪代码，然后填入配置：

theme: gungnirTheme({
  themePlugins: {
    ba: "your-tracking-code"
  }
})
