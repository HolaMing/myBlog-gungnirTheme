1 min

信息

默认不启用该功能。

将 hitokoto 项设置为 true 可以开启首页的一言气泡，将鼠标悬浮在头像上气泡就会显示出来：

theme: gungnirTheme({
  hitokoto: true
})

如果你想更改一言的请求参数，或想使用其他的接口，可以：

theme: gungnirTheme({
  hitokoto: "https://v1.hitokoto.cn?c=i"  // 只返回诗词
})
在 GitHub 上编辑此页
