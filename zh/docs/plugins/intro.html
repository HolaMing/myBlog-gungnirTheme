在写这个主题的过程中，为了实现一些功能，我顺便写了一些插件，因此我把它们的文档都放在这一节。

这是插件列表：

  • plugin-giscus：Giscus（基于 GitHub Discussions 的评论系统）
  • plugin-chart：Chart.js（图表）
  • plugin-mermaid：Mermaid（流程图等）
  • plugin-katex：KaTeX（公式渲染）
  • plugin-md-plus：给 Markdown 添加更多语法和功能，目前支持脚注和高亮标记
  • plugin-reading-time：字数统计和阅读时长估计
  • plugin-baidu-tongji: 百度统计
