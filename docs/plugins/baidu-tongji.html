plugin-baidu-tongji

Richer
2021-12-27
2 min

Plugin @renovamen/vuepress-plugin-baidu-tongji@next for adding 百度统计 to VuePress 2. It is edited from @vuepress/plugin-google-analytics.

# Install

pnpm install @renovamen/vuepress-plugin-baidu-tongji@next

yarn add @renovamen/vuepress-plugin-baidu-tongji@next

npm install @renovamen/vuepress-plugin-baidu-tongji@next

# Usage

const { baiduTongjiPlugin } = require("@renovamen/vuepress-plugin-baidu-tongji");

module.exports = {
  plugins: [
    baiduTongjiPlugin({
      "id": "your-tracking-code"
    })
  ]
}

id is your tracking code offered by 百度统计.
